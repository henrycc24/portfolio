## Course Overview

ECE 4160 Fast Robots covers embedded sensing, motor control, and real-time robot control on a mobile platform.

## Robot Platform

Lab work spans the course robot platform used for control, sensing, and embedded software exercises.

## Labs / Projects

Course labs build toward integrated robot behavior including sensor processing, control loops, and embedded software modules. See the linked GitHub site for lab writeups and results.

## Sensors and Control

Work includes interfacing sensors, implementing control algorithms, and validating robot behavior through structured lab milestones.

## Lessons Learned

Tight integration between sensing, control timing, and embedded software is critical for reliable mobile robot behavior.

## Future Work

Continue integrating course control concepts into research robot platforms.
